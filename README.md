# 104

##Slide's Name
使用西元日期命名資料夾，e.g.
```
slides/20121221/
```
並將slide本身命名為index，e.g.
```
slides/20121221/index.html
```
或在減號後加上關鍵字，e.g.
```
slides/20121221-Doom/index.html
```
其餘範例檔案、附件檔案與圖片、影片等多媒體資料也放置於同一資料夾中。
