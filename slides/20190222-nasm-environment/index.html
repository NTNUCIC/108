sudo apt install build-essential

終端機基本指令

基本指令

  • cd 切換路徑
  • ls 查看當前目錄檔案
  • chmod 更改檔案權限(777 -rwxrwxrwx)
  • su 轉為super user(系統管理員)
  • whoami 確認目前身分
  • uname -a 系統資訊
