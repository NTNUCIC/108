建構子

可以傳入參數並且建立物件的函式

使用this關鍵字設定物件的屬性

對於建立自訂物件很重要

搭配new關鍵字建立物件

踹踹看


function Cat(name,age,color){
  this.name=name
  this.age=age
  this.color=color
  this.toString=function(){
    return name+" is a "+age+" old "+color+" cat"
  }
}

this關鍵字

代表是誰擁有此段程式碼

在函式中被使用則代表該擁有該函式的物件

在物件中被使用代表該物件

在建構子中代表了新的物件

這個this也是JS神奇的部份...
