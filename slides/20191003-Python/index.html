何謂程式

簡單來說: y = f(x) 其中 x 是資料, y 是資訊, f 是程式

像是一本教電腦如何做菜(我們要的東西)的食譜

  • 給電腦遵循的一套規則 (SOP)
  • 可以快速/大量生產 (資訊)
  • 無須了解實作細節 (黑箱)

讓我們回到開始的問題上... 何謂程式語言?

如同英語, 法語, 日語, ...等

依靠一定的單字, 文法, 語句

讓人們得以有所依靠建立溝通的橋樑

程式語言就是指用來描述程式的規則

依靠一定的關鍵字與結構語句

讓人們能與電腦建立溝通的橋樑

Python語言的優缺點

print("Hello World !")
  • 優點
    • 易學習, 可讀性佳
    • 開源社群, 大量第三方資源
  • 缺點
    • 弱型別語言, 執行期錯誤
    • 速度較慢, 移動計算較弱

輸出 / 輸入

變數與資料型態

變數

  • Python的變數不具有型態
  • 用英文開頭, 可以英數字混用
  • 命名規則以能讓人理解為重

資料型態

  • 整數(int): 1, 2, 3, ...
  • 浮點數(float): 0.3, 2.7, ...
  • 字串(str): 'Hello World', ...
  • 布林(bool): True, False

運算符

  • 括號 ()
  • 加 +
  • 減 -
  • 乘 *
  • 除 /
  • 除-取商數 //
  • 除-取餘數 %
  • 次方 **
